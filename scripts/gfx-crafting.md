# Gfx Crafting

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-crafting klasörünü resources/ dizinine kopyalayın
cp -r gfx-crafting /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-crafting
```
