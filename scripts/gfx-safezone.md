# Gfx Safezone

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-safezone klasörünü resources/ dizinine kopyalayın
cp -r gfx-safezone /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-safezone
```
