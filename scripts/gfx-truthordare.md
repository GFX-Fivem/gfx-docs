# Gfx Truthordare

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-truthordare klasörünü resources/ dizinine kopyalayın
cp -r gfx-truthordare /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-truthordare
```
