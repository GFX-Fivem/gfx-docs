# Gfx Shop

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-shop klasörünü resources/ dizinine kopyalayın
cp -r gfx-shop /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-shop
```
