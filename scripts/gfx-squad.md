# Gfx Squad

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-squad klasörünü resources/ dizinine kopyalayın
cp -r gfx-squad /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-squad
```
