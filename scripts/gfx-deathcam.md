# Gfx Deathcam

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-deathcam klasörünü resources/ dizinine kopyalayın
cp -r gfx-deathcam /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-deathcam
```
