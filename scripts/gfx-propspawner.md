# Gfx Propspawner

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-propspawner klasörünü resources/ dizinine kopyalayın
cp -r gfx-propspawner /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-propspawner
```
