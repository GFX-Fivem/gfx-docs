# Gfx Help

## Kurulum

### 1. Dosyaları Kopyala
```bash
# gfx-help klasörünü resources/ dizinine kopyalayın
cp -r gfx-help /path/to/resources/
```

### 2. server.cfg
```cfg
ensure gfx-help
```
